Monitor SSL/TLS certificate expiry for all XProtect™ HTTPS endpoints
Open source — no warranty, use at own risk Auto-check every 6 hours

How It Works

The Certificate Watchdog plugin automatically discovers all HTTPS endpoints in your XProtect installation — servers and devices — and monitors their SSL/TLS certificate expiry dates.

When certificates approach expiry, XProtect analytics events are fired at configurable thresholds (60, 30, and 15 days). A Smart Client workspace dashboard provides a live overview of all certificate statuses.

Getting Started

  1. The plugin works out of the box — no configuration required.
  2. Open the Certificates workspace tab in the Smart Client to view all endpoint statuses.
  3. Use the Recollect button to trigger an immediate certificate re-check.

Smart Client Dashboard

The Certificates workspace tab shows two sortable tables for server and device certificates. Click any column header to sort.

  • Recollect — Triggers a full certificate re-check on the Event Server (endpoint discovery, certificate validation, and event firing). Use after replacing a certificate.
  • Refresh — Fetches the latest cached results from the Event Server without triggering a new check.

Permissions

The Smart Client dashboard is controlled by role-based security. In Management Client under Security > Roles, the plugin appears as “Certificate Watchdog” with a Read permission. Grant it to allow the Certificates workspace tab; deny or leave unset to hide it. Event Server background checks and event firing are not affected.

Architecture

The plugin consists of three components that run in different processes:

Management Client (Admin UI)

Provides the configuration interface you are using now. The plugin node appears in the sidebar. Configuration is minimal — the plugin auto-discovers all HTTPS endpoints.

Event Server (Background Plugin)

Runs as a background service inside the Milestone Event Server. Discovers HTTPS endpoints, checks certificate validity, and fires analytics events at the configured expiry thresholds. Checks run automatically every 6 hours and within 20 seconds of configuration changes.

Smart Client (Workspace Plugin)

Provides the Certificates workspace tab. Displays certificate status for all server and device endpoints. Communicates with the Event Server to fetch results and trigger re-checks.
