Subscribe to a metadata channel and inspect the live ONVIF event stream
Open source - no warranty, use at own risk Mgmt Client only

What this plugin does

Metadata Viewer is a debugging tool for ONVIF metadata streams. Pick any metadata channel exposed by your XProtect site, subscribe live, and watch each NotificationMessage arrive as a row with parsed topic, timestamps, and pretty-printed XML for inspection.

Use it when you're wiring up analytic events from a camera, validating an ONVIF integration, or trying to figure out which topic a device actually publishes on.

Usage

  1. Right-click Metadata Viewer in the tree and choose Create New.
  2. Name the viewer and pick a metadata channel from the picker.
  3. Click Start to subscribe. Each incoming ONVIF NotificationMessage becomes a row; newest appears on top.
  4. Click a row to see the pretty-printed, syntax-highlighted XML in the preview pane below.
  5. Click Stop when done, or Clear to empty the table.

Columns

ColumnDescription
Seq#Per-viewer sequence number, assigned on arrival.
Capture Time (UTC)From the ONVIF tt:Message/@UtcTime. A trailing * means the attribute was missing and the column shows the receive time instead.
Event Topic TreeThe wsnt:Topic value, e.g. tnsaxis:CameraApplicationPlatform/ALPV.Update.
DataCompact XML of the NotificationMessage. Click the row to see the full payload highlighted below.
InfoName=Value pairs extracted from tt:SimpleItem entries. Items found under tt:Source are shown in square brackets, e.g. [channel=1].

Filter

Type a substring and press Apply (or Enter). Matches Topic, Data, and Info case-insensitively. Clear empties the filter.

Notes

  • Payloads that aren't ONVIF NotificationMessage events (or fail to parse) are dropped into a single row with the raw XML in the Data column so nothing is lost.
  • The viewer runs in the Management Client only; no Event Server component, no Rules integration.
  • If the Start button stays disabled after picking a channel, the selected item is not a valid metadata source.
  • Diagnostic messages (subscribe details, heartbeats, status flags, errors) go to %ProgramData%\Milestone\XProtect Management Client\Logs\MIPtrace.log; search for MetadataViewer.
