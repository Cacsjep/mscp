ZIP Downloads - Readme

When you download a ZIP from this page on a different machine, Windows attaches a "Mark of the Web" flag and Smart Client / Recording Server may refuse to load the DLLs inside. Right-click the ZIP, choose Properties, tick Unblock, then extract.

Downloads done on this management server itself (via http://localhost/mscp/) are not flagged.

PKI Cert Installer

v__VERSION__

Standalone single-file Windows EXE. Connects to this management server, lists every PKI-plugin certificate, and installs the picked ones into the local Windows certificate store with the right private-key ACLs. Run this on each XProtect™ server (Recording / Management / Mobile / Event / Failover) you want to deploy certs to. Requires admin (UAC).
