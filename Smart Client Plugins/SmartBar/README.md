# Smart Bar

A workspace toolbar plugin for Milestone XProtect Smart Client. Adds a button to the toolbar in Live and Playback workspaces.

## Installation

Copy the build output to:

```
C:\Program Files\Milestone\MIPPlugins\SmartBar\
```

Restart the Smart Client.

## Features

- Workspace toolbar button in Live and Playback mode
- FontAwesome icon via CommunitySDK
