# Reporting Bugs

Found a bug? We appreciate you taking the time to report it. Good bug reports help us fix issues faster.

## Where to Report

All bugs are tracked on GitHub Issues:

**[Open a new issue](https://github.com/Cacsjep/mscp/issues/new)**

## Community Discord

You can also reach us on the **Milestone Developer Community** Discord server. Ask questions, discuss features, or get help with plugin issues.

**[Join Discord](https://discord.gg/EU2HX4Qc)**
